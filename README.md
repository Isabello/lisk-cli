# Lisk CLI

A command line interface for bootstrapping and managing [Lisk](https://lisk.io/) dapps.

## Installation:

```
npm install -g lisk-cli
```

## Dapps

### Add dapp

```
lisk-cli dapps -a
```

### Deposit

```
lisk-cli dapps -d
```

### Withdraw

```
lisk-cli dapps -w
```

## Contracts

### Add contract

```
lisk-cli contract -a
```

### Delete contract

```
lisk-cli contract -d
```

## Crypto

### Generate public key

```
lisk-cli crypto -p
```

### Generate accounts

```
lisk-cli crypto -g
```

### Help

```
lisk-cli -h
```
